# 1. Introdução

[Próximo: Objetivos](./02-Objetivos.md)

O avanço das tecnologias da informação e dos sistemas baseados em dados tem impulsionado a criação de aplicações capazes de organizar grandes volumes de informações e oferecer experiências mais personalizadas aos usuários. Em ambientes digitais modernos, a integração entre diferentes tipos de banco de dados, arquiteturas de software e interfaces web tem se tornado fundamental para garantir desempenho, escalabilidade e usabilidade.

Nesse contexto, sistemas que lidam com conteúdo multimídia, interação social e produção contínua de dados demandam estruturas robustas que permitam armazenar, consultar e relacionar informações de maneira eficiente.

No cenário atual, o consumo de mídias como filmes, séries, músicas e livros ocorre de forma intensa e descentralizada. Usuários interagem com diversas plataformas e aplicativos, porém muitas dessas experiências acabam dispersas e não são organizadas de forma integrada. Surge, assim, a necessidade de um sistema que permita registrar, avaliar, organizar e compartilhar mídias de maneira centralizada, estruturada e social.

O sistema MyZone foi desenvolvido como resposta a esse problema, oferecendo uma plataforma web completa para gerenciamento e avaliação de mídias. A aplicação combina um banco de dados relacional MySQL, responsável por armazenar informações estruturadas e críticas para o funcionamento do sistema, com um banco NoSQL MongoDB, utilizado para a gestão de avaliações, que demandam maior flexibilidade e alta frequência de escrita.

O backend, implementado em Java com Spring Boot, garante robustez e segurança na comunicação entre os bancos de dados e o frontend, enquanto a interface construída com HTML, CSS e JavaScript possibilita a interação intuitiva do usuário com as funcionalidades do sistema.
