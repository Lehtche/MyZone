# 2. Objetivos

[Anterior: Introdução](./01-Introducao.md) | [Próximo: Metodologia](./03-Metodologia.md)

A presente seção apresenta as metas que orientaram o desenvolvimento do sistema MyZone, definindo tanto o propósito central do projeto quanto as ações específicas necessárias para sua implementação.

### 2.1 Objetivo Geral

Desenvolver uma plataforma web integrada para organização, avaliação e compartilhamento de mídias, combinando banco de dados relacional e NoSQL a fim de garantir desempenho, flexibilidade e consistência no armazenamento das informações.

### 2.2 Objetivos Específicos

* Modelar e implementar um banco de dados relacional MySQL com tabelas, relacionamentos, índices, triggers, views e procedimentos armazenados.
* Implementar um banco de dados NoSQL MongoDB para armazenamento flexível e escalável das avaliações.
* Construir um backend em Java com Spring Boot, responsável por autenticação, controle de acesso, regras de negócio e integração entre os bancos.
* Desenvolver uma interface web funcional utilizando HTML, CSS e JavaScript para interação direta com o usuário.
* Implementar operações de CRUD para as principais entidades do sistema, incluindo gerenciamento de usuários, mídias e coleções.
* Garantir a integridade, a segurança e o desempenho das operações por meio de boas práticas de desenvolvimento, controle de permissões e organização arquitetural.
