# 8. Guia de Execução

[Anterior: Banco NoSQL](./07-Banco-NoSQL-MongoDB.md) | [Próximo: Conclusão](./09-Conclusao-e-Referencias.md)

Este guia descreve os passos necessários para configurar e executar o projeto MyZone localmente.

### Pré-requisitos

* JDK 21 ou superior.
* Maven 3.x.
* Servidor MySQL (recomendado 8.0) a correr em `localhost:3306`.
* Servidor MongoDB (recomendado 7.0) a correr em `localhost:27017`.
* Uma chave de API do [The Movie Database (TMDb)](https://www.themoviedb.org/documentation/api) (é gratuita).
* (Opcional) Uma chave de API do Google Books.
